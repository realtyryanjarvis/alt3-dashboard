Equipment Room

$25,552.64
Scope of Work
  1. 42U Skeleton Rack Frame (Strong) — Installation of the 42U Skeleton Rack Frame in a suitable location within the room. This involves ensuring that the floor can support the weight and that there is enough clearance for ventilation and access. Mounting of the frame following the manufacturer's guidelines. Positioning the frame to allow for efficient cable management.
  2. IP UPS Kit - 12 Controllable Outlets | 1100 VA (Wattbox) — Installation of the IP UPS Kit within the rack frame. Connection of the UPS to the power supply and integration with the network to enable remote management. Configuration of the UPS settings, including power failure actions and alerts.
  3. Router (Araknis) — Installation of the router within the rack frame. Connection of the router to the network and power supply. Configuration of the router settings as per the network requirements. This includes setting up security parameters, SSID, passwords, and IP settings.
  4. Araknis 16 Port Partial POE Switch — Installation of the POE Switch within the rack frame. Connection of the switch to the router and power supply. Configuration of the switch settings, including VLAN setup and optimization of POE settings.
  5. Araknis Networks WiFi Access Points — Installation and configuration of the Araknis Networks devices within the rack frame. Connection of the devices to the POE switch and power supply. Configuration of the devices as per the network requirements.
  6. AVP-AC-AEX-RC-HUB (AVPro Edge) — Installation of the AVP-AC-AEX-RC-HUB within the rack frame. Connection of the hub to the network and power supply. Configuration of the hub and integration with the network.
  7. Sonos Amp G1 (×6) — Installation of the Sonos Amp G1 units within the rack frame. Connection of the amps to the power supply and speaker outputs. Configuration of the amps through the Sonos app, including integration with the network and setting up zones.
  8. Josh Core: Hardware + Lifetime Software Plan — Installation of the Josh Core within the rack frame. Connection of the device to the network and power supply. Configuration of the device through the Josh.ai interface, including integration with the network and other devices.
  9. Araknis SFP+ Transceiver Module — Installation of the SFP+ Transceiver Module within the rack frame. Connection of the module to the network switch/router. Configuration of the module settings as per the network requirements.
  10. Cat6 Cabling (Vanco/1TouchSolutions) — Installation of the Cat6 cables within the rack frame and room. Connection of the cables to the respective devices. Testing of the cables for continuity, speed, and signal strength.
  11. Audio Interconnects (Binary) — Installation of the audio interconnects within the rack frame. Connection of the interconnects to the Sonos Amp G1 units and speakers. Testing of the interconnects for audio quality and signal strength.
  12. DMARK (1TouchSolutions) — Installation of the DMARK within the rack frame. Connection of the DMARK to the network and power supply. Configuration of the DMARK settings as per the network requirements.

Testing and Commissioning: Once all devices are installed and configured, a comprehensive testing and commissioning process will be performed. This includes: checking the physical installation for safety and compliance with manufacturer's guidelines; performing a network test to ensure all devices are communicating correctly; testing the UPS for correct operation during a power failure; checking the router and switch settings for security and performance; performing a sound check for the Sonos Amp G1 units and speakers; running a system test for the Josh Core device; ensuring all cables are functioning correctly and securely connected; verifying the correct operation of the DMARK; documenting all configurations and settings for future reference. Includes 4 hours of programming integrations.
